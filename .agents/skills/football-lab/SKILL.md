# スキル: FOOTBALL LAB データ取得

## サイト構造の概観

FOOTBALL LAB（`https://www.football-lab.jp`）のページは大きく3種類に分かれる。

| 種別 | 説明 | 主なURL |
|------|------|---------|
| チームページ | 各クラブのデータ（チームスタイル・試合・個人CBPランキング等） | `/{チームコード}/...` |
| 選手ページ | 選手個人の成績・CBP推移・プレースタイル指標 | `/player/{player_id}` |
| リーグサマリー | 全チーム横断のランキング・比較データ | `/summary/...` |

データ更新は**主に月・火曜夜**。マッチレポートも同様のタイミング。

## URL 規則一覧

### チームページ（`/{チームコード}/サブページ`）

| サブページ | URL例（大分） | 内容 |
|------------|---------------|------|
| チームスタイル（トップ） | `/oita/` | チームスタイル指標・攻撃パターン分布 |
| 試合一覧 | `/oita/match?year=2026` | 年間の試合日程・結果リスト。各行にマッチレポートへのリンクあり |
| マッチレポート | `/oita/report?year=2026&month=03&date=21` | 試合詳細スタッツ・CBP・時間帯別データ |
| 個人CBPランキング | `/oita/ranking` | チーム内選手の節別・累計CBPランキング |
| シーズンサマリー | `/oita/summary` | シーズン通算のチーム・個人成績 |
| シーズン比較 | `/oita/season_comparison` | 過年度との指標比較（2014年〜） |
| 移籍情報 | `/oita/transfer` | 加入・放出選手一覧 |
| 対戦シミュレーション | `/oita/simulation` | イベントデータによる対戦予測 |

**マッチレポート**は**WebFetch で直接取得することを優先**（検索より正確）。その他のチームサブページも積極的にFetchする。

### 選手ページ

```
https://www.football-lab.jp/player/{player_id}
```

- `player_id` は数値ID（例: `1300038`）
- 選手のIDは **試合ページ・個人CBPランキングページのリンク**から取得するか、以下の検索クエリで確認する:
  - `{選手名} site:football-lab.jp/player`
  - `{選手名} football-lab 選手ページ`
- ページ内容: 節別の個人成績（ゴール・アシスト・CBP各種）推移、プレースタイル指標、類似選手

### リーグサマリーページ

```
https://www.football-lab.jp/summary/{カテゴリ}/{j1|j2|j3}?year=YYYY[&data={指標キー}]
```

| カテゴリ | 指標キー | 取得できる内容 |
|----------|----------|---------------|
| `team_ranking` | `chance` | チャンス構築率・シュート成功率ランキング |
| `team_ranking` | `kagi` | AGI・KAGIランキング |
| `team_ranking` | `goal` | 得点パターン内訳ランキング |
| `team_ranking` | `possession` | ボール保持率ランキング |
| `team_ranking` | （なし） | 総合順位表 |
| `player_parameter` | `1`〜`15` | プレースタイル指標別の選手ランキング |
| `cb_point` | — | チームCBPランキング（URLはFetchで確認） |

**例**: 2026年J2のチャンス構築率
```
https://www.football-lab.jp/summary/team_ranking/j2?year=2026&data=chance
```

## FOOTBALL LAB 固有指標

以下に指標名の一覧を示す。**詳細な定義・計算方法・公式解説URL**は `references/metrics.md` を参照すること。

### チームレベル指標
- チャンス構築率、攻撃回数、30メートルライン進入回数
- AGI（Approach Goal Index）、KAGI（Keep Away from Goal Index）
- ゴール期待値（xG）

### CBP（チャンスビルディングポイント）
- 攻撃CBP、守備CBP、奪取CBP
- パスCBP、パスレシーブCBP、クロスCBP、ドリブルCBP
- シュートCBP、ゴールCBP、セーブCBP

### チームスタイル指標（16種類、J2はトラッキングなしのため8種類）
- セットプレー、左サイド攻撃、右サイド攻撃、中央攻撃
- ショートカウンター、ロングカウンター、ポゼッション、セーフティ
- コンパクトネス（J1のみ）、最終ライン（J1のみ）、ハイプレス（J1のみ）

### プレースタイル指標（選手個人・11項目20段階）
- 決定力、シュート力、パスレスポンス力、パスチャンス力
- クロスチャンス力、ドリブルチャンス力、ビルドアップ力
- 敵陣空中戦、自陣空中戦、守備力、ボール奪取力

## データ取得の優先戦略

```
マッチレポート    → WebFetch（URL直接取得）を最優先
チームサブページ  → WebFetch（URL直接構築）を優先
選手ページ       → IDを先に特定してからWebFetch
リーグサマリー   → WebFetch（URL直接構築）を優先
上記で取得不可   → WebSearchにフォールバック
```

## 検索クエリ設計パターン

### 試合単体
- `大分トリニータ football-lab マッチレポート {YYYY}年{MM}月{DD}日`
- `大分トリニータ {相手チーム名} {YYYY}/{MM}/{DD} スタッツ`
- `site:football-lab.jp oita report {YYYY}`

### シーズン推移・平均
- `大分トリニータ CBP {YYYY} シーズン平均`
- `大分トリニータ football-lab {YYYY} チームスタイル`
- `大分トリニータ チャンス構築率 {YYYY}`

### リーグ比較・ランキング
- `J2 守備CBP ランキング {YYYY}`
- `J2 チャンス構築率 ランキング {YYYY} football-lab`
- `J2 AGI KAGI {YYYY} football-lab`
- `J2 {指標名} ランキング {YYYY} football-lab`

### 個人データ
- `{選手名} football-lab CBP {YYYY}`

- `大分トリニータ 個人CBP ランキング {YYYY}`
- `{選手名} football-lab {YYYY} プレースタイル`

## データ取得の注意事項

1. **支持データと反証データの両方を探す** — 都合のいいデータだけ集めない
2. **今節値と今季平均を両方取得** — 今節値だけでは文脈がわからない
3. **データが見つからない場合は「データなし」と明記** — 推測で補わない
4. **FOOTBALL LABのデータは必ずURLを出典として記載**
5. **他試合（別節）のデータを混入させない**
6. **J2はトラッキングデータ非対象のため、コンパクトネス・最終ライン・ハイプレスなどのチームスタイル指標は取得不可**
7. **マッチレポートの更新タイミングは主に月・火曜夜** — 直近試合で取得できない場合は未更新の可能性がある
