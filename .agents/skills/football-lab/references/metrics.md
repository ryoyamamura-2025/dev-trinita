# FOOTBALL LAB 指標定義リファレンス

各指標の概念・読み方を記載する。**厳密な計算式・公式定義**が必要な場合は各セクション末尾のURLをWebFetchで参照すること。

## CBP（チャンスビルディングポイント）

**公式解説**: `https://www.football-lab.jp/pages/cb_point`

CBP = 試合を通じてシュートチャンスの構築にどれだけ貢献したかを数値化した指標。シュートに到達した攻撃を遡り、関与したプレーのエリア・難易度・シュート到達率をもとにポイントを算出する。守備CBPは相手の攻撃を阻止した際に攻撃側のポイントがそのまま守備側に付与される。

| 指標名 | 意味 |
|--------|------|
| **攻撃CBP** | パス・クロス・ドリブルCBPの合計。攻撃でチャンスを作った量・質 |
| **守備CBP** | 守備で相手のチャンスを潰した量・質 |
| **奪取CBP** | ボール奪取でチャンスに繋げた量・質 |
| **パスCBP** | パス（出し手）でチャンスを作った量・質 |
| **パスレシーブCBP** | パス（受け手）でチャンスを作った量・質 |
| **クロスCBP** | クロスでチャンスを作った量・質 |
| **ドリブルCBP** | ドリブルでチャンスを作った量・質 |
| **シュートCBP** | シュートでチャンスに繋げた量・質 |
| **ゴールCBP** | ゴールへの直接貢献度 |
| **セーブCBP** | GKがシュートを防いだ貢献度 |

## KAGI / AGI

**公式解説**: `https://www.football-lab.jp/pages/kagi`

| 指標名 | 定義・読み方 |
|--------|------------|
| **AGI（Approach Goal Index）** | 攻撃の際にどれだけ相手ゴールに近づけたかを示す指標。①攻撃時間のうち相手ゴールに近い位置でボールを保持していた時間の割合、②攻撃開始から敵陣ペナルティエリア到達までの時間の短さ、の2軸で算出 |
| **KAGI（Keep Away from Goal Index）** | 守備の際に相手を自陣ゴールに近づけなかったかを示す指標（AGIの守備版）。①相手攻撃時間のうち自陣ゴールから遠い位置でボールを保持させていた時間の割合、②相手の攻撃開始から自陣ペナルティエリア到達までの時間の長さ、で算出 |

## チームスタイル指標

**公式解説**: `https://www.football-lab.jp/pages/team_style`

攻撃の種類ごとに「指数」（リーグ内偏差値）と「シュート率」（その攻撃がシュートに到達した確率）の2軸で構成される。

**データ制約**: J2・J3はトラッキングデータ非対象のため、以下のうちトラッキングデータを要する指標（コンパクトネス・最終ライン・ハイプレス・一部のスタイル）は**取得不可**。

| スタイル | 概要 | データ |
|----------|------|--------|
| セットプレー | アタッキングサードでのFK・CK・スローインからの攻撃 | J1/J2/J3 |
| 左サイド攻撃 | 左サイドを経由した攻撃 | J1/J2/J3 |
| 右サイド攻撃 | 右サイドを経由した攻撃 | J1/J2/J3 |
| 中央攻撃 | 中央を経由した攻撃 | J1/J2/J3 |
| ショートカウンター | ボール奪取から素早く攻撃に転じた速攻（短距離） | J1/J2/J3 |
| ロングカウンター | 自陣深くからのロングカウンター | J1/J2/J3 |
| ポゼッション（敵陣） | 相手の守備ブロックを崩す遅攻 | J1/J2/J3 |
| セーフティ（自陣） | 自陣からのビルドアップ型の攻撃 | J1/J2/J3 |
| コンパクトネス | 守備時の陣形の縦・横の幅（密集度） | **J1のみ** |
| 最終ライン | 守備ブロックの重心の高さ | **J1のみ** |
| ハイプレス | 高い位置での守備プレス強度 | **J1のみ** |

## プレースタイル指標（選手個人）

**公式解説**: `https://www.football-lab.jp/pages/playing_style`

各選手のイベントデータから11項目を20段階評価。全体の1/3以上出場した選手が対象（90分換算）。

| 項目 | 概要 |
|------|------|
| **決定力** | シュート決定率とシュートCBPの合成値 |
| **シュート力** | シュートCBP（量・質） |
| **パスレスポンス力** | パスの受け手としての貢献（パスレシーブCBP） |
| **パスチャンス力** | パスの出し手としての貢献（パスCBP） |
| **クロスチャンス力** | クロスでのチャンス創出（クロスCBP） |
| **ドリブルチャンス力** | ドリブルでのチャンス創出（ドリブルCBP） |
| **ビルドアップ力** | 自陣からのパス・ボール運搬貢献 |
| **敵陣空中戦** | 相手陣内での競り合い勝率・頻度 |
| **自陣空中戦** | 自陣での競り合い勝率・頻度 |
| **守備力** | 守備CBP（相手チャンスを潰した量・質） |
| **ボール奪取力** | 奪取CBP（ボール奪取からチャンスに繋げた量・質） |

## ゴール期待値（xG）

**公式解説**: `https://www.football-lab.jp/pages/expected_goal`

| 項目 | 内容 |
|------|------|
| **定義** | シュートが得点に結びつく確率を0〜1で表した指標。FOOTBALL LABはJリーグのデータをもとに独自算出 |
| **目安** | 0.2以上がいわゆる「決定機」水準 |
| **活用** | チームや選手の実力を「結果」ではなく「チャンスの質と量」で評価するために使う |

## チームレベル指標（補足）

| 指標名 | 定義・読み方 |
|--------|------------|
| **チャンス構築率** | 攻撃回数のうちシュートに到達した割合（シュート数 ÷ 攻撃回数）。決定力とは別に「どれだけ崩せているか」を示す |
| **攻撃回数** | 自チームが攻撃フェーズに移行した総回数。ポゼッション量ではなくトランジションの多さを反映する |
| **30メートルライン進入回数** | 相手ゴールから30m以内のエリアへ侵入した回数。アタッキングサードへの到達頻度を示す |
