# /edit — Phase 3: 編集・最終保存フェーズを実行

`articles/` の最新下書きをチェックリストで検証・修正し、最終保存して完了レポートを表示します。

## 前提確認

実行前に `articles/` ディレクトリに下書きファイルが存在することを確認してください。
ファイルがない場合は `/write` を先に実行してください。

## 実行手順

以下の手順で実行してください:

1. `.claude/agents/editor-agent.md` を Read ツールで読む
2. そのファイルの内容を prompt として、**Agent ツール（subagent_type: "general-purpose"）** を起動する
   - エージェントはメインの会話とは独立したコンテキストで動作する
3. エージェントが返した完了レポートをそのままユーザーに表示する

## エージェントから完了レポートを受け取ったら

以下の形式で表示してください（エージェントが生成した値をそのまま使う）:

```
✅ 記事生成完了
ファイル: articles/{YYYYMMDD}_{相手チーム名}.md
文字数: {XXXX}字
データ引用数: {XX}件
出典: FOOTBALL LAB マッチレポート {YYYY-MM-DD}
```
